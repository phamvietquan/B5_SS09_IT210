# Bài 5: Trải nghiệm Đa ngôn ngữ (I18N)

## 1. Thiết kế kiến trúc và luồng hoạt động

### Luồng hoạt động (Data Flow)

1. Người dùng truy cập website
2. Người dùng chọn ngôn ngữ qua URL (ví dụ: ?lang=ja hoặc ?lang=ko)
3. Spring Interceptor sẽ "đánh chặn" request và đọc tham số lang
4. Hệ thống cập nhật Locale tương ứng (ja, ko, en)
5. LocaleResolver lưu lại ngôn ngữ đã chọn
6. Thymeleaf sử dụng Resource Bundle để render nội dung theo ngôn ngữ
7. Khi người dùng quay lại:
    - Hệ thống lấy Locale đã lưu
    - Tự động hiển thị đúng ngôn ngữ trước đó

---

## 2. Công cụ Spring sử dụng

### LocaleChangeInterceptor

- Đây là thành phần "đánh chặn" request
- Tự động đọc parameter "lang" trên URL

Ví dụ:
?lang=ja
?lang=ko

- Sau đó thay đổi Locale tương ứng cho request hiện tại

---

### Resource Bundle (messages.properties)

- Lưu các bản dịch

Ví dụ:
messages.properties (mặc định - tiếng Anh)
messages_ja.properties (tiếng Nhật)
messages_ko.properties (tiếng Hàn)

- Thymeleaf dùng key để hiển thị text:
  #{cart} → Giỏ hàng / カート / 장바구니

---

## 3. Lựa chọn LocaleResolver

### Chọn: CookieLocaleResolver

---

### Lý do

- Cookie được lưu trên trình duyệt người dùng
- Có thể thiết lập thời gian sống dài (ví dụ: 30 ngày)
- Khi người dùng quay lại website:
    - Cookie vẫn còn
    - Hệ thống tự động lấy lại ngôn ngữ đã chọn

=> Đáp ứng yêu cầu:
"Không cần chọn lại ngôn ngữ ở lần truy cập sau"

---

### Tại sao không dùng SessionLocaleResolver?

- Session sẽ mất khi:
    + Tắt trình duyệt
    + Hết timeout

=> Không đáp ứng yêu cầu ghi nhớ lâu dài

---

## 4. Xử lý bẫy dữ liệu (Fallback)

Trường hợp:
?lang=ar (ngôn ngữ không hỗ trợ)

Cách xử lý:
- Cấu hình default locale = English
- Nếu không tìm thấy file messages tương ứng:
  → Spring tự động fallback về messages.properties

=> Không bị lỗi trắng trang

---

## 5. Kết luận

- Sử dụng LocaleChangeInterceptor để bắt request đổi ngôn ngữ
- Dùng Resource Bundle để quản lý đa ngôn ngữ
- Chọn CookieLocaleResolver để lưu cấu hình lâu dài
- Thiết lập default locale để đảm bảo fallback an toàn
